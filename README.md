# Offline messager Yii2 #

## Установка ##

Выполнить команду для установки зависимостей

```
composer install
```

Выполнить инициализацию нужного окружения

```
php init
```

Заполнить локальные конфиги верным доступом к бае данных, настроить домен на директорию web

Выполнить миграции

```
yii migrate
```
